# RentCarApp

Este é um projeto simples de um sistema de aluguel de carros desenvolvido em Python.

## Descrição

O projeto consiste em um programa que permite ao usuário cadastrar carros com informações como modelo, marca, cor e ano. Além disso, é possível visualizar os carros cadastrados.

## Funcionalidades

- Adicionar novos carros ao sistema com informações detalhadas.
- Visualizar a lista de carros cadastrados.
- Funciona via linha de comando.

## Proximos Updates

- Cadastros de perfis para alugar
- Colocar um sistema para alugar os carros cadastrados
